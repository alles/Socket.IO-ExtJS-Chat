# Socket.IO + ExtJS Chat.

A simple chat for socket.io + extjs

## Dependencies

You need install [Sencha CMD](https://www.sencha.com/products/extjs/cmd-download/).

## How to use

...
$ cd test-task
$ npm install
$ npm start
$ cd public
$ sencha app build
...


And point your browser to `http://localhost:3000`.
